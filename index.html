Mission ATLAS — Physics Games Portfolio

Mission ATLAS is a browser-based collection of astronomy-themed physics games designed for classroom use. Each mission turns a scientific idea into a playable challenge: observe, calculate, test, fail safely, and try again. The correct physical model is the most reliable path to success.

Mechanics, gravitation, energy, circuits, waves, data analysis and modern physics — all framed through space exploration, observation, and decision-making. The games below link directly to the HTML files in this repository.

How to use this portfolio

  • Open a mission directly from the cards above.
  • Use the games as warm-ups, mini-labs, revision activities, or homework missions.
  • Each file is standalone HTML, so deployment and classroom sharing stay simple.

Design principles

  • Physics first: formulas, units, and model choices are visible.
  • Astronomy as anchor: every game is framed by a defensible space-science context.
  • Anti-randomness: feedback rewards reasoning, not blind slider spam.
